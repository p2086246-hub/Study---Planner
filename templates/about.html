About SmartStudy AI

B.Tech 6th Semester Mini Project — AI/ML Based Smart Study Planner

Project Overview

SmartStudy AI is a full-stack web application that uses Machine Learning to help students plan their study schedule intelligently. Instead of manually deciding how many hours to spend on each subject, the system automatically predicts this using a Linear Regression model trained on key features.

Timetable Features

  • Full 7-day school-style timetable (MON–SUN)
  • 1 Revision slot at the end of every day
  • 1 Mock Test slot every Saturday
  • Smart hour boost when exams are within 14 days
  • Priority-weighted slot distribution

ML Module

  • Algorithm: Linear Regression (Manual OLS)
  • Features: Difficulty, Days Left, Concentration
  • Training: 600-sample synthetic dataset
  • Accuracy: R² score shown on Dashboard

Adaptive Logic

  • Exam within 7 days + low hours → auto-boost up to +2 hrs/day
  • Streak system resets if all sessions not completed
